# 📊 CEAPS 2021 - Análise Exploratória de Dados

## Sobre o projeto

Este projeto apresenta uma análise exploratória dos gastos do CEAPS (Cota para o Exercício da Atividade Parlamentar dos Senadores) referentes ao ano de 2021.

O objetivo foi transformar dados públicos em informações relevantes por meio de técnicas de limpeza, tratamento, análise e visualização de dados utilizando Python.
