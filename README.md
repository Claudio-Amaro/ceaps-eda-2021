# 📊 CEAPS 2021 – Análise Exploratória de Dados (EDA)

## 📌 Sobre o Projeto

Este projeto apresenta uma Análise Exploratória de Dados (Exploratory Data Analysis - EDA) dos gastos do **CEAPS (Cota para o Exercício da Atividade Parlamentar dos Senadores)** referentes ao ano de **2021**.

O objetivo foi transformar dados públicos em informações relevantes por meio de técnicas de limpeza, tratamento, agregação e visualização de dados, permitindo identificar padrões de utilização dos recursos públicos destinados à atividade parlamentar.

O projeto foi desenvolvido utilizando **Python**, **Pandas** e **Plotly**, com foco em análise de dados, visualização e construção de indicadores.

---

## 🎯 Objetivos

- Realizar a limpeza e preparação dos dados.
- Desenvolver uma Análise Exploratória de Dados (EDA).
- Construir indicadores (KPIs).
- Criar gráficos interativos.
- Identificar padrões e tendências.
- Produzir insights baseados em dados.

---

## 🛠 Tecnologias Utilizadas

- Python
- Pandas

---

## 📊 Base de Dados

Foram analisados os dados públicos do CEAPS referentes ao exercício de **2021**, disponibilizados pelo Senado Federal.

A base contém informações sobre:

- Senadores;
- Categorias de despesas;
- Fornecedores;
- Datas dos reembolsos;
- Valores reembolsados;
- Documentos fiscais.

---

## 📈 Principais Resultados

- 16.827 registros analisados.
- 86 senadores avaliados.
- 2.991 fornecedores identificados.
- Aproximadamente R$ 25 milhões em reembolsos analisados.
- Cinco categorias concentraram praticamente todo o orçamento executado.

---

## 💡 Principais Insights

A análise mostrou que as despesas com **consultorias** representaram a maior parcela dos reembolsos realizados em 2021.

Também foi observado que cinco categorias de despesas concentraram praticamente todo o orçamento analisado, destacando a importância de consultorias, passagens, locomoção, aluguel de escritórios e divulgação da atividade parlamentar.

Esses resultados evidenciam um padrão de concentração dos gastos e demonstram o potencial da análise exploratória para apoiar a compreensão da utilização dos recursos públicos.
